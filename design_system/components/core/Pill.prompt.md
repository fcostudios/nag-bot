Role / label pill — an italic chip with full pill radius, from the "bienvenidas" (welcome) pieces where it carries a job title.

```jsx
<Pill>Consultor de RRHH</Pill>
<Pill variant="filled" color="var(--qph-area-salud)">Salud</Pill>
<Pill italic={false}>Activo</Pill>
```

`variant`: `outline` (default) or `filled`. `color` sets the accent (use an area color when relevant).
