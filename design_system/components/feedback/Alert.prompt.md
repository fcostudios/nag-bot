Inline alert — subtle gray panel with a colored left border and status dot.

```jsx
<Alert tone="success" title="Éxito">Comunicado publicado correctamente.</Alert>
<Alert tone="warning" title="Atención">Actualiza el diseño cada 6 meses.</Alert>
<Alert tone="info">Usa el color del área correspondiente.</Alert>
```

Tones: `success` · `warning` · `danger` · `info`.
